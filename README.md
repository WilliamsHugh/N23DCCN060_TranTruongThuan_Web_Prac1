# [Thực hành 1] Product Listing Page - NextJS & Tailwind

**Họ tên:** Trần Trường Thuận
**MSSV:** N23DCCN060

## Mô tả
Bài thực hành ứng dụng web hiển thị danh sách sản phẩm lấy từ FakeStoreAPI, xây dựng bằng NextJS App Router và Tailwind CSS.

## Tính năng chính
- Trang danh sách sản phẩm responsive (mobile -> desktop)
- Trang chi tiết sản phẩm với dynamic route '/product/[id]'
- Fetch dữ liệu thực tế từ fakestoreapi.com

## Cách chạy
- B1: npm install
- B2: npm run dev
